# 再生回数マイルストーン — 操作説明

## 初回セットアップ

### 1. YouTube Data API キーの取得

1. [Google Cloud Console](https://console.cloud.google.com/) にアクセス
2. プロジェクトを作成（または既存を使用）
3. 「APIとサービス」→「ライブラリ」→「YouTube Data API v3」を有効化
4. 「認証情報」→「認証情報を作成」→「API キー」
5. 取得したキーをコピー

> 無料枠（日次クォータ 10,000 単位）で十分です。

### 2. ツールの初回起動

1. ページを開くと API キー入力画面が表示される
2. 取得したキーを入力して「保存して開始」をクリック
3. 自動的に動画データの取得が始まる

## 動画カード

各カードには以下の情報が表示されます：

| 要素 | 説明 |
|------|------|
| サムネイル | クリックで YouTube 動画ページを開く |
| 再生時間 | サムネイル右下にオーバーレイ表示 |
| ★ボタン | カード右上。クリックでマーク ON/OFF（金色になるとマーク済み） |
| 再生数 | 現在の再生回数 |
| 勢い | 再生数の右側に「+XX万/日」形式で表示。直近72時間の増加速度 |
| NEXT MILESTONE | 次の節目再生数・残り再生数・予測到達日・進捗バー |
| 投稿日 | 投稿年月日と経過日数（例: 投稿: 2024/3/15 · 120日前） |
| 他チャンネル | 手動追加した他チャンネル動画に青いバッジを表示 |
| 🐦 ツイートボタン | タイトル・URL・再生数情報を含むツイート画面へ遷移 |

### 予測到達日（ETA）の表示形式

| 状態 | 表示例 |
|------|-------|
| 「もうすぐ」判定の動画 | `今日 23:45` / `明日 08:30` / `5/20 14:00` |
| その他の動画 | `約14日後` / `明日` / `今日中！` |

### カードの色分け

| 色 | 意味 |
|----|------|
| 通常（ダークブルー） | 特に変化なし |
| オレンジ枠 | 近日中（設定した日数以内）に節目到達予定 |
| ゴールド枠・発光背景 | 過去48時間以内に節目を達成 |
| 薄ゴールド枠 | 過去72時間以内に節目を達成（48h超） |

## ツールバー操作

### フィルタ

| ボタン | 動作 |
|--------|------|
| 全て | すべての動画を表示 |
| ★のみ | マーク済みの動画のみ表示 |

### ソート

| ボタン | 動作 |
|--------|------|
| 再生数 | 再生数の多い順 |
| 投稿日 | 投稿日の新しい順 |
| 注目 | **達成済み**・**もうすぐ**・**その他** の3セクションに分け表示 |

#### 「注目」タブのセクション構成

| セクション | 対象 | 内部ソート |
|-----------|------|----------|
| 達成済み | 72時間以内に節目達成 | 48h以内達成を優先、その後再生数降順 |
| もうすぐ | 設定した日数以内に到達予測 | ETA 昇順（近い順） |
| その他 | 上記以外 | 再生数降順 |

---

## 🐦 ツイートボタンの内容

動画の状態に応じてツイート文が自動生成されます：

| 状態 | ツイート例 |
|------|-----------|
| 48h以内に達成 | `タイトル URL 【100万再生達成🎉】` |
| もうすぐ達成 | `タイトル URL まもなく100万再生！（あと3.2万）` |
| 通常 | `タイトル URL 36.7万再生` |

---

## 🔔 ブラウザ通知

1. ヘッダーの 🔔 ボタンをクリック
2. ブラウザの許可ダイアログで「許可」を選択
3. 自動更新のタイミングで、**★マーク済みの動画**が新たに節目を達成すると通知が届く

> **注意:** ページを開いている間のみ通知が届きます。タブを閉じると通知されません。  
> 通知を受け取りたい動画は事前に ★ をつけておいてください。

## ⚙ 設定

| 項目 | 説明 | デフォルト |
|------|------|-----------|
| YouTube Data API Key | API キー（localStorage にのみ保存） | — |
| チャンネルハンドル | 対象チャンネルの @ 以降の文字列 | `OtonoseKanade` |
| 「もうすぐ」閾値（日） | 何日以内の予測到達をオレンジ表示するか | `3` |
| 自動更新間隔（分） | タブを開いている間の自動取得間隔 | `30` |

### 他チャンネル動画の追加

設定画面の「動画URLを追加」テキストエリアに YouTube の動画 URL を貼り付けて「追加して取得」をクリックすると、他チャンネルの動画も追跡対象に加えられます。

- 複数の URL を複数行でまとめてペーストできる
- 追加済み動画は「追加済み動画」リストに表示され、「削除」ボタンで個別に除外できる
- 追加した動画は通常チャンネル動画と同様に、自動更新ごとに再生数が更新される
- 他チャンネルの動画には「他チャンネル」バッジが表示される

### データ管理

| ボタン | 動作 |
|--------|------|
| JSON 出力 | 現在の全データ（動画履歴・設定・マーク・ピン留め）を JSON ファイルとしてダウンロード |
| JSON 取り込み | 出力した JSON ファイルを読み込み、このブラウザのデータと置き換え |
| データ削除 | 動画履歴・マーク・通知記録をリセット（API キーと設定は保持） |

> **JSON 出力/取り込みの用途:** 別のブラウザや端末にデータを移行したり、バックアップを取るために使用します。

---

## データの蓄積について

- 視聴履歴は **7日間** 保持されます
- 初回起動直後は履歴が1点のみのため「勢い計算中...」と表示され、予測ETAも算出されません
- タブを開き続けるほど精度が向上します
- データはこのブラウザの localStorage にのみ保存されます

## よくある質問

**Q. 別のVTuberにも使えますか？**  
A. 設定のチャンネルハンドルを変更することで対応可能です。データ削除してから切り替えることを推奨します。

**Q. API キーが漏れる心配はありますか？**  
A. キーはこのブラウザの localStorage にのみ保存され、外部サーバーには送信されません。YouTube API への直接リクエストにのみ使用されます。

**Q. ショート動画やライブ配信も表示されますか？**  
A. 表示されます。手動マーク機能（★）で追跡したい動画だけに絞り込むことができます。

**Q. 別端末でも同じデータを見たい**  
A. 設定画面の「JSON 出力」でデータを書き出し、別端末で「JSON 取り込み」することでデータを移行できます。

**Q. 他チャンネルの動画も一緒に管理したい**  
A. 設定画面の「動画URLを追加」に YouTube URL を貼り付けて追加できます。奏ちゃんが参加している合唱動画やコラボ楽曲などを追加すると、同じ画面でまとめて確認できます。
