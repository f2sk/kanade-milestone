# 再生回数マイルストーン — 操作説明

## 初回セットアップ

### 1. YouTube Data API キーの取得

1. [Google Cloud Console](https://console.cloud.google.com/) にアクセス
2. プロジェクトを作成（または既存を使用）
3. 「APIとサービス」→「ライブラリ」→「YouTube Data API v3」を有効化
4. 「認証情報」→「認証情報を作成」→「API キー」
5. 取得したキーをコピー

> 無料枠（日次クォータ 10,000 単位）で十分です。

### 2. ツールの初回起動

1. ページを開くと API キー入力画面が表示される
2. 取得したキーを入力して「保存して開始」をクリック
3. 自動的に動画データの取得が始まる

## 動画カード

各カードには以下の情報が表示されます：

| 要素 | 説明 |
|------|------|
| サムネイル | クリックで YouTube 動画ページを開く |
| 再生時間 | サムネイル右下にオーバーレイ表示 |
| ★ボタン | カード右上。クリックでマーク ON/OFF（金色になるとマーク済み） |
| 再生数 | 現在の再生回数 |
| NEXT MILESTONE | 次の節目再生数・残り再生数・予測到達日・進捗バー |
| 勢い | 直近72時間の1日あたり増加数 |
| 🐦 ツイートボタン | タイトル・URL・再生数情報を含むツイート画面へ遷移 |

### カードの色分け

| 色 | 意味 |
|----|------|
| 通常（ダークブルー） | 特に変化なし |
| オレンジ枠 | 近日中（設定した日数以内）に節目到達予定 |
| ゴールド枠＋アニメーション | 過去48時間以内に節目を達成 |

## ツールバー操作

### フィルタ

| ボタン | 動作 |
|--------|------|
| 全て | すべての動画を表示 |
| ★のみ | マーク済みの動画のみ表示 |

### ソート

| ボタン | 動作 |
|--------|------|
| 再生数 | 達成(48h) → もうすぐ → その他（再生数降順） |
| 投稿日 | 投稿日の新しい順 |
| もうすぐ | 次のマイルストーン到達予測日が近い順（ETA昇順）。データ蓄積中の動画は末尾 |

---

## 🐦 ツイートボタンの内容

動画の状態に応じてツイート文が自動生成されます：

| 状態 | ツイート例 |
|------|-----------|
| 48h以内に達成 | `タイトル URL 【100万再生達成🎉】` |
| もうすぐ達成 | `タイトル URL まもなく100万再生！（あと3.2万）` |
| 通常 | `タイトル URL 36.7万再生` |

---

## 🔔 ブラウザ通知

1. ヘッダーの 🔔 ボタンをクリック
2. ブラウザの許可ダイアログで「許可」を選択
3. 自動更新のタイミングで新たに節目を達成した動画があると通知が届く

> **注意:** ページを開いている間のみ通知が届きます。タブを閉じると通知されません。

## ⚙ 設定

| 項目 | 説明 | デフォルト |
|------|------|-----------|
| YouTube Data API Key | API キー（localStorage にのみ保存） | — |
| チャンネルハンドル | 対象チャンネルの @ 以降の文字列 | `OtonoseKanade` |
| 「もうすぐ」閾値（日） | 何日以内の予測到達をオレンジ表示するか | `3` |
| 自動更新間隔（分） | タブを開いている間の自動取得間隔 | `30` |

**データ削除**: 設定画面下部の「データ削除」ボタンで視聴履歴・マーク・通知記録をリセットします（API キーと設定は保持）。

---

## データの蓄積について

- 視聴履歴は **7日間** 保持されます
- 初回起動直後は履歴が1点のみのため「勢い計算中...」と表示され、予測ETAも算出されません
- タブを開き続けるほど精度が向上します
- データはこのブラウザの localStorage にのみ保存されます（他端末とは同期されません）

## よくある質問

**Q. 別のVTuberにも使えますか？**  
A. 設定のチャンネルハンドルを変更することで対応可能です。データ削除してから切り替えることを推奨します。

**Q. API キーが漏れる心配はありますか？**  
A. キーはこのブラウザの localStorage にのみ保存され、外部サーバーには送信されません。YouTube API への直接リクエストにのみ使用されます。

**Q. ショート動画やライブ配信も表示されますか？**  
A. 表示されます。手動マーク機能（★）で追跡したい動画だけに絞り込むことができます。
